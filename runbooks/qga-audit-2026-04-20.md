## Notes

- **qga-installed**: reflects Proxmox VM config (`agent=enabled=1` or `agent=1` in config API)
- **qga-active**: reflects live qga socket response (only meaningful for running VMs)
- **guest-exec**: whether `guest-exec` command is in qga's supported_commands and enabled
- **stopped-vm**: VM is stopped; qga status cannot be verified live but config shows it is enabled
- LXC containers do not support qemu-guest-agent
- Template VMs (9xxx series) excluded from critical scope
